# Changelog

All notable changes to this project will be documented in this file.

- Project README and updated app graphics.

- Incorrect time adjustment behavior when changing time to below 10 seconds.

## [1.0] - 2024-11-19

### Added
- Initial release of Game Clock.
- Core timer functionality with pause and adjust capabilities.
